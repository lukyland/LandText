# LandText

A lightweight text editor for Windows.

## Features
- **Multi-Window Support** - Open multiple editor windows simultaneously
- **File Operations** - New, Open, Save, Save As with unsaved changes detection
- **Undo/Redo** - Full undo/redo support with keyboard shortcuts
- **Find** - Search text with highlighting and auto-scroll to results
- **Themes** - Light, Dark, and fully customizable Custom theme
- **Font Customization** - Adjustable font size (8-32pt) with live preview
- **Persistent Settings** - All preferences saved automatically

## Themes

### Built-in Themes
- **Light** - Classic light theme
- **Dark** - Modern dark theme
- **Custom** - Create your own theme with full color control

### Using Themes
1. Go to **Settings > Theme**
2. Select a theme (Light, Dark, or Custom)
3. Click **Apply Theme**

### Customizing Colors
1. Go to **Settings > Theme**
2. Select **Custom**
3. Click color pickers to customize:
   - Background
   - Text Color
   - Selection Background
   - Selection Text
   - Status Bar Background
   - Status Bar Text
4. Click **Save Colors** to preserve your custom theme
5. Click **Apply Theme** to apply to all open windows

## Font Size
1. Go to **Settings > Font Size**
2. Drag the slider (range: 8-32pt)
3. Click **Apply** to change font in all windows

## Multi-Window
Open multiple files in separate windows:
1. Go to **Window > Open New Window**
- Theme and font size changes apply to all windows
- Each window independently tracks its own file and modifications

## File Support
- Text files (.txt)
- All text-based file formats
- UTF-8 encoding

## System Requirements
Windows 7 or higher
